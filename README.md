# 🤖 Automated AI Analytics Assistant

A robust automated pipeline built to connect LLMs directly to structured data, minimizing token bloat while delivering instant data insights.

## 🛠️ Tech Stack
* **Language:** Python
* **Environment:** Jupyter Notebook / Production Script
* **Libraries:** Pandas, OpenAI / Anthropic API, LangChain

## 🚀 Setup & Installation
1. Clone this repository: `git clone https://github.com/Bharath-AI-Modules/ai-analytics-assistant.git`
2. Install dependencies: `pip install pandas openai langchain`
3. Create a `.env` file and add your credentials: `OPENAI_API_KEY=your_key_here`
